**REX-RAG: Reasoning Exploration with Policy Correction in Retrieval-Augmented Generation**

A reinforcement learning framework for RAG that escapes reasoning dead ends through a **mixed sampling strategy** and maintains stable policy learning via a principled **policy correction mechanism**. Delivers significant performance boosts on multi-hop reasoning and general QA tasks with strong out-of-domain generalization.
